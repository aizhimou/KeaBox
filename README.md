# KeaBox 🦜

**KeaBox** is a curated collection of lightweight, single-page web utilities designed to solve specific digital pain points with elegance and simplicity.

The project is named after the **Kea**, an alpine parrot endemic to New Zealand. Renowned for their extraordinary intelligence and problem-solving skills, Kea are famous for their curiosity and "tool-using" behavior. This repository aims to embody that same spirit: providing smart, efficient tools to dismantle everyday technical friction.

---

## 🛠 Included Tools

### Netscape Cookie Visualizer (`/web-tools/cookie-viewer`)
A zero-dependency, client-side tool to parse and visualize Netscape HTTP Cookie files (often used by `curl`, `wget`, or browser extensions).
- **Features**:
    - **Visual Parsing**: Converts raw tab-separated text into a clean, searchable table.
    - **Time Conversion**: Automatically transforms Unix timestamps into readable `yyyy-MM-dd HH:mm:ss` format.
    - **Smart Sorting**: Support for sorting by Expiry date to easily identify session vs. persistent cookies.
    - **Security Awareness**: Visual tags for `HttpOnly` and `Secure` flags.
- **Tech Stack**: Pure HTML5, CSS3, and Vanilla JavaScript.

## 🚀 How to Use

All tools in this repository are designed to be "plug-and-play":

1.  **Clone the repository**
2.  **Open in Browser**:
    Navigate to the specific tool directory (e.g., `web-tools/cookie-viewer/index.html`) and open the file in any modern web browser. 
    *No server or installation required.*

## 🦜 Why KeaBox?

In the wild, Kea are known for their ability to use objects as tools to get what they want. As developers, we often need "digital screwdrivers" to quickly inspect, convert, or visualize data without jumping through complex software hoops. 

**KeaBox** is where these "digital screwdrivers" live.
